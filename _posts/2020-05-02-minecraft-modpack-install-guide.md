## 개요

운영체제가 윈도우고 마인크래프트가 이미 깔려있다고 가정한다.

트위치를 통한 모드팩 설치는 기존 마인크래프트와 전혀 별개의 공간을 사용한다.

처음에 실행파일을 받아오기 위해 한번 실행해줘야 하는 것이 끝이며, 그 이후에는 서로에게 영향을 미치지 않는다.

## 설치법

### 1. 트위치를 설치한다

[트위치 다운로드 링크](https://www.twitch.tv/downloads)

위의 링크로 들어가 윈도우로 설치한다.

### 2. Mods(모드)탭으로 들어간다

로그인을 하고 모드탭에 들어가서 마인크래프트를 활성화 한다.

주의:

* **마인크래프트 경로는 기존 마인크래프트 경로가 아닌 다른 빈 경로로 지정한다**
* **마인크래프트 경로 중 한글이 들어가선 안된다**

실행이 잘 될 경우 3-1로, 오류가 난 경우 3-2로 간다.

### 3-1. 실행이 잘 된 경우

모드팩 탐색 탭에 들어간 뒤 지정된 모드팩을 깔아준다.

트위치의 내 모드팩에 있는 모드팩에서 플레이버튼을 통해 실행한다.

추가할 모드가 없다면 끝.

#### 추가할 모드가 있다면

내 모드팩에 있는 모드팩을 클릭해준다.

오른쪽 상단의 톱니바퀴를 클릭한 후 폴더 열기를 누른다.

업로드된 파일중 `additional_mods`를 받은 뒤 내용물을 확인한다.

mods 폴더와 config 폴더를 열린 폴더에 바로 복사해준다.

### 3-2. 실행중 오류가 난 경우

트위치를 지워도 된다.

기본 마크 폴더로 간다.

업로드된 파일 중 `no twitch`를 받은 후 내용물을 확인한다.

폴더들의 이름과 같은 마인크래프트 내 폴더들을 삭제하고, 파일의 폴더들을 복사해준다.

기본 마크를 실행한다.

### 4. 모드 추가

#### 트위치를 통해 깐 경우

내 모드팩에 있는 모드팩을 클릭해준다.

오른쪽 상단의 톱니바퀴를 클릭한 후 폴더 열기를 누른다.

업로드된 파일 중 .jar 파일을 mods 폴더에 넣고 .cfg 파일을 config 폴더에 넣는다.

#### 기본 마크폴더에 깐 경우

기본 마인크래프트 폴더에 간다.

업로드된 파일 중 .jar 파일을 mods 폴더에 넣고 .cfg 파일을 config 폴더에 넣는다.
