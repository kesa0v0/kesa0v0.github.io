## 오늘 한 것

오늘은 [HOIEDITOR](https://github.com/kesa0v0/HOIEditor)를 좀 더 개선했다.

원래는 leaders 에디터을 만드려 했으나 어제 만든 Conditions 에디터를 좀 더 개선하는데 하루를 모두 써 버렸다.

----

### 개선점

1. 타이틀을 추가
2. 우클릭 매뉴 추가
3. 더 긴 Row랑 더 큰 폰트

----

#### 1. 타이틀을 추가

Form -> Conditions

Form -> Conditions Edit

----

#### 2. 우클릭 매뉴 추가

![우클릭 매뉴](/assets/images/2020-04-29/2020-04-29-right-click.png)

* Add Trigger : 트리거 추가
* Edit Trigger : 트리거 수정
* Remove Trigger : 트리거 삭제

* Cut : 자르기  *(미완성)*
* Copy : 복사  *(미완성)*
* Paste : 붙여넣기  *(미완성)*

----

#### 3. 더 긴 Row랑 더 큰 폰트

폰트를 키우니 Row의 크기도 커졌다. 흠터레스팅

※ **기본으로 적혀있는 예시는 적용되지 않음**

----

### 개선할 점

1. Cut, Copy, Paste 기능 완성
2. 트리거 검색 기능 추가
